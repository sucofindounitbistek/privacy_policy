# Privacy Policy - IDSurvey

Repositori ini berisi halaman statis untuk Kebijakan Privasi (Privacy Policy) layanan IDSurvey.

## Deskripsi

Halaman ini dibuat menggunakan HTML dan CSS dengan gaya desain modern (*glassmorphism*, responsif, dan mendukung mode gelap / *dark mode*). Halaman ini dirancang agar mudah dimuat, elegan, dan ramah pengguna di berbagai perangkat.

## Struktur File

- `index.html`: Berisi struktur halaman HTML beserta *styling* CSS internal dan sedikit *script* JavaScript untuk efek animasi pada bagian *header* saat pengguna melakukan *scroll*.

## Cara Menjalankan

Karena ini adalah file statis HTML, tidak ada instalasi khusus (seperti Node.js atau *framework*) yang diperlukan. Anda hanya perlu membuka file `index.html` menggunakan *web browser* modern (Google Chrome, Mozilla Firefox, Safari, Edge, dll).

Jika Anda ingin memodifikasi teks, warna, atau desain, Anda dapat mengedit langsung pada file `index.html`.
